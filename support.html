How to Play

Super Decoder is a logic-based color code-breaking game. A secret 4-color code is hidden at the start of each round — your goal is to crack it within 7 attempts using deduction and the feedback from each guess.

Start a Game

Tap NEW to begin. A secret 4-color code is randomly generated from 7 possible colors.

Fill Your Row

Tap each empty cell in the active row to pick a color. You must fill all 4 cells before submitting.

Submit Your Guess

Once all 4 cells are filled, tap SUBMIT. Your guess is instantly analyzed.

Read the Feedback

Colored bars appear below each cell — use them to refine your next guess logically.

Win or Lose

Get all 4 green bars to win. Run out of rows (7 attempts) without cracking the code and it's game over.

Play Again

Tap PLAY AGAIN on the result screen or hit NEW at any time to start fresh.

// reading the clues

Feedback System

After each submitted row, a small indicator bar appears beneath every cell. These bars are your only clues — learn to read them quickly.

Green bar — Correct color, correct position. This cell is locked in.

White bar — Right color, but wrong position. It belongs somewhere else in the code.

No bar / dark — This color does not appear anywhere in the secret code.

Tip: Feedback is given per position — a white bar on cell 2 doesn't tell you which position the color belongs in, only that it's somewhere else. Use process of elimination across rows to narrow it down.

FAQ

Can the same color appear twice in the code?
No. The secret code always uses 4 unique colors chosen from the 7 available. You'll never see the same color repeated in a single code.
How many colors are in the game?
There are 7 colors total: Red, Green, Orange, Purple, Yellow, Blue, and Pink. The secret code uses 4 of these 7 each game, chosen at random.
How many attempts do I get?
You have 7 attempts per game. The attempt counter at the bottom of the screen tracks how many guesses you've used. If you reach row 7 without cracking the code, the game ends and the secret code is revealed.
What does the counter at the bottom show?
The digital display shows your current attempt number out of 7 — e.g. 002 / 7 attempts means you've submitted 2 guesses. The colored dots beside it show all 7 available colors for reference.
Can I change a color after placing it?
Yes — as long as you haven't submitted the row yet, simply tap any cell in the active row again to change its color. Once you hit SUBMIT, that row is locked.
How do I start a new game mid-round?
Tap the NEW button in the top-right corner at any time. This immediately resets the board and generates a fresh secret code.
Is there a timer or time limit?
No. Super Decoder is completely untimed — take as long as you need to think through your strategy.
The app isn't behaving as expected. What should I do?
Try closing and reopening the app first. If the issue persists, make sure you're running the latest version from the App Store. If it still occurs, please contact us at ian.jyz@gmail.com with a description of what happened.
